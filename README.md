# 🛒 Food Cart Program (Python)

## 📌 Project Description

This is a simple command-line based food cart application built using Python.
The program allows users to add food items along with their prices and calculates the total bill amount.

This project was developed to practice Python programming concepts such as:

* Lists (Data Structures)
* While loops
* User input handling
* Basic billing logic
* Running total calculation

---

## 🚀 Features

* Add multiple food items to cart
* Enter price for each food item
* Quit option to stop adding items
* Displays all selected food items
* Calculates and displays total bill amount

---

## 🛠️ Technologies Used

* Python 3
